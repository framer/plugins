# Airtable Plugin

Plugin for syncing Airtable tables into the Framer CMS.

**By:** @sakib25800, @triozer, and @benjamindenboer

![Airtable Image](../../assets/airtable.png)

## Development

To use a local OAuth worker during development, set the `VITE_LOCAL` environment variable:

```bash
VITE_LOCAL=true yarn dev
```

You can use a different URL if you provide your own worker. See the [OAuth guide](https://www.framer.com/developers/oauth) for more information.
